01.数组

array

  • go里面array表示的是一片连续的内存
  • 再声明的时候就已经确定数组的大小
  • 函数内部使用的是栈内存, 函数参数传递的时候会进行整个内存的拷贝

示例代码

func call_arr(numbers *[0x888]int) {}
func main() {
    arr_int := [0x888]int{}
    arr_int[1] = 2
    call_arr(arr_int)
}
  • main函数里面的直接提升栈空间
  push    rbp
  mov     rbp, rsp
  sub     rsp, 8920h
  movups  *(rsp+8920h+var_10), xmm15
  • 作为参数传递的时候是直接复制的堆栈
  mov     rdi, rsp
  lea     rsi, [rsp+8920h+var_44E0]
  mov     ecx, 888h
  rep     movsq
  call    main_call_arr
  • 使用指针作为函数参数的时候就是直接使用的堆栈地址
