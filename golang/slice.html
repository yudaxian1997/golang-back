02.切片

slice

  • go里面的动态数组，初始化的时候不能确定其长度大小
  • 底层使用 runtime_makeslice 来初始化切片
  • 切片的本质是一个包含长度和容量的结构体
type SliceHeader struct {
    Data uintptr
    Len  int
    Cap  int
}
  • 再函数内部，或者传值的时候切片数据也是保存再堆栈上面
  • 使用栈来保存数组的地址， 再扩容的时候修改栈的地址

示例代码

func do_arr(arr []int) {
    // 这里修改无效，因为只是修改栈上面的数据
    arr = append(arr, 2)
}

func show_arr(arr []int) {
    for _, v := range arr {
        fmt.Println(v)
    }
}

func main() {
    // 初始化的时候不能指定长度
    // 本质是结构体，保存再栈上面
    arr := []int{}
    // 添加元素，可能会修改底层数组
    arr = append(arr, 1)
    show_arr(arr)
    // 参数传递，值传递
    do_arr(arr)
    show_arr(arr)
}
